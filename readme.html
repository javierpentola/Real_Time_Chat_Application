Real-Time Chat Application

Project Description

This project is a real-time chat application created with HTML, CSS, JavaScript, Node.js, and WebSocket. It allows instant communication between users, updating the chat in real time as messages are sent and received.

Main Features

User Interface:

  • Designed and structured with HTML and CSS, using the 98.css library for a retro Windows 98 style.
  • Client-side logic managed with JavaScript to connect with the server and update the chat.

Backend:

  • Node.js server with WebSocket to handle real-time connections and messages.
  • Message storage in a JSON file.

Testing and Deployment:

  • Using ngrok to expose the local server through a public URL for external testing.
  • Dynamic request for the WebSocket URL from the user.

Key Learnings

  • WebSocket: Establishing and managing real-time connections between client and server.
  • Node.js: Creating a server and managing files.
  • Frontend Development: Creating interactive and stylized user interfaces.
  • Integration and Testing: Using tools like ngrok for dynamic testing and deployment.

How to Run the Project

Clone the repository:

git clone <repository>
cd chat-app

Install dependencies:

npm install

Start the server:

node server/server.js

Start ngrok:

ngrok http 3000

Open the application:

Access the public URL provided by ngrok in your browser.

Enter the requested WebSocket URL.
